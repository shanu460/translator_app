This is a modern, fast and friendly Translator built by Ishaan. It supports typing or speaking (STT) and speaks back results (TTS) in Male/Female voices where available. Multiple Indian languages are supported including Hindi, Punjabi, Bengali, Gujarati, Marathi, Tamil, Telugu and Urdu.

Frontend uses glassmorphism UI with smooth animations. The backend is built in Flask and translation uses googletrans. Voice features run in the browser via the Web Speech APIs.
